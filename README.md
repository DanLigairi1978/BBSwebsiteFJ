## 🚀 Quick Start
1. Clone: `git clone https://github.com/DanLigairi1978/BBSwebsiteFJ.git`
2. Open `index.html` in browser or run local server
3. Make changes and push to GitHub - auto-deploys in 1-2 minutes

## 📂 Structure
- `index.html` - Home page
- `services.html` - Services page
- `why-us.html` - Why Us page
- `resources.html` - Resources page
- `contact.html` - Contact page
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images, icons, fonts

## 🎨 Design
- Mobile-first responsive design
- Fast loading (< 3s on 3G)
- Colors: Deep Teal, Warm Coral, Cream
- Fonts: Poppins (headings), Inter (body)
